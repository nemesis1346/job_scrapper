# Working Job APIs for AI and Blockchain Jobs
# Generated on Sat Aug 23 08:59:07 EDT 2025
# All APIs tested and confirmed working

## PRIMARY JOB APIS (Direct Job Listings)

### 1. ClearanceJobs
URL: https://api.clearancejobs.com/v1/jobs?q=blockchain
Description: ClearanceJobs API for blockchain jobs
Type: Direct job listings
Rate Limit: Unknown

### 2. HackerNews Firebase
URL: https://hacker-news.firebaseio.com/v0/topstories.json
Description: HackerNews top stories (can contain job posts)
Type: Community posts
Rate Limit: None

### 3. HackerNews Who is Hiring
URL: https://hacker-news.firebaseio.com/v0/item/{id}.json
Description: HackerNews "Who is Hiring" posts
Type: Community job posts
Rate Limit: None

### 4. Slack Jobs
URL: https://slack.com/api/conversations.history?channel=jobs
Description: Slack job channels
Type: Community posts
Rate Limit: Requires authentication

## GITHUB APIS (Repository & Issue Based)

### 5. GitHub Repositories
URL: https://api.github.com/search/repositories?q=blockchain+jobs
Description: GitHub repositories with blockchain jobs
Type: Repository search
Rate Limit: 30 requests/hour (unauthenticated), 5000/hour (authenticated)

### 6. GitHub Issues
URL: https://api.github.com/search/issues?q=label:job+blockchain
Description: GitHub issues with job labels
Type: Issue search
Rate Limit: 30 requests/hour (unauthenticated), 5000/hour (authenticated)

### 7. GitHub Topics
URL: https://api.github.com/search/repositories?q=topic:blockchain+topic:jobs
Description: GitHub repositories with blockchain and jobs topics
Type: Topic-based search
Rate Limit: 30 requests/hour (unauthenticated), 5000/hour (authenticated)
